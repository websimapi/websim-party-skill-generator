Party Skill Plugin

RuneLite Extension

Features

  • 🎈 Balloon Tracking: Tracks single (3xp), double (6xp), and triple (9xp) pops.
  • 📊 Virtual Levels: Custom 1-99 leveling curve.
  • ⛵ Sailing Replacement: Overrides the 24th skill slot (Sailing) with Party.
  • 📦 Ready to Build: Includes Gradle wrapper and Readme.
